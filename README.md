# ElectriX

ElectriX est une application Windows fournie sous forme d'executable. Le depot contient l'application prete a lancer ainsi qu'un rapport de projet au format PDF.

## Contenu du depot

- `ElectriX.exe` : application Windows executable.
- `ElectriX_Project_Report.pdf` : rapport de projet.
- `docs/INSTALLATION.md` : guide rapide pour telecharger et lancer l'application.
- `docs/RELEASE_CHECKLIST.md` : checklist avant une nouvelle publication.

## Installation rapide

1. Ouvrir la page du depot GitHub.
2. Telecharger `ElectriX.exe`.
3. Lancer le fichier sur Windows.

Si Windows affiche un avertissement SmartScreen, verifier d'abord que le fichier vient bien de ce depot officiel avant de choisir l'option d'execution.

## Compatibilite

L'application est fournie pour Windows. Les autres systemes d'exploitation ne sont pas pris en charge par cet executable.

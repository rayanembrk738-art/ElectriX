# Installation

## Prerequis

- Un ordinateur sous Windows.
- Le fichier `ElectriX.exe` telecharge depuis le depot officiel.

## Lancer l'application

1. Aller sur la page GitHub du projet.
2. Cliquer sur `ElectriX.exe`.
3. Telecharger le fichier.
4. Double-cliquer sur `ElectriX.exe` pour demarrer l'application.

## Avertissement Windows

Windows peut afficher un avertissement si l'application n'est pas signee. Dans ce cas :

1. Verifier que le fichier provient bien de ce depot.
2. Cliquer sur `Informations complementaires`.
3. Choisir `Executer quand meme` si vous faites confiance a la source.

## Probleme au lancement

Si l'application ne s'ouvre pas :

- Verifier que le telechargement est complet.
- Deplacer l'executable dans un dossier simple, par exemple `Documents` ou `Bureau`.
- Relancer l'application en tant qu'administrateur si necessaire.
